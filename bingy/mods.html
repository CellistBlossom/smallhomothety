How do I mod Hollow Knight?

Modding Hollow Knight is really easy! Firstly, if you'd like to install debug mod, or downpatch for a speedrun, please refer to the Hollow Knight Speedrun.com Page.

First, install the Hollow Knight Modding API. Download the zip file in the releases page, and drag the contents into your Hollow Knight's /hollow_knight_Data/Managed/ folder. This information can also be found in the zip file's Readme.md.

Finally, you can either manually install mods by downloading the individual mods and placing them in the /Managed/Mods folder, or by installing Lumafly (formerly known as "Scarab+") or Scarab!
Scarab or Lumafly are Hollow Knight's dedicated Mod Installers. They include a large database of Hollow Knight mods that you can download, and it will directly place it into your mods folder. They can be found at the links below.

Required Mods for Bingy

Note: If you install mods via Scarab or Lumafly, all dependencies will be automatically installed.
If you are not using Lumafly or Scarab, it is very important to read what each mods' dependencies are. Remember to download them along with the mod if installing them manually.

Benchwarp A mod that allows you to warp to and from any bench that you've previously rested at. Bingy uses the warp-only feature of this mod.

BingoAdvancedCustomGeneration A mod that adds the Tournament 3 goalpack to BingoSync generation options, and line exclusions which are not natively supported by the BingoSync mod.

BingoGoalPack1 A mod that allows automarking for extended goals and more, some of which are used in the Tournament 3 goalpack.

BingoSync Allows for joining rooms (the password is always "fast") & board generation from within the game, as well as automarking for goals in the base vanilla Hollow Knight goalpack.

BingoUI Adds an overlay that keeps track of progress towards certain goals.

QoL A Quality of Life mod that includes many different modifications to the game that make it more fun for speedrunning. This includes bringing back inventory drops, lever skips and various other glitches, as well as instant text and menuing to make bingy faster.

Additional Mods

These are mods that aren't required for most bingy games, but may be desired or required in certain situations.

BaldurRngConfig A mod to standardize baldur spits. Not tournament-legal, and currently nonstandard, so ask opponents before using it in competitve matches.

BingoBoardReplay Echoes marks from a board in one room to a board in another room, used in tournaments to allow for commentator delay.

BingoGoalPack2 A mod that adds automarking for some additional goals, not used in any of the most common bingy goalpacks but used for some others.

DebugMod Various utilities. Useful for practice or testing, but not legal for use in most bingy games unless format-specific rules allow it. If DebugMod is installed when starting a bingy, make sure to lock keybinds first.

LoadNormalizer A mod used for competitive matches, usually lockout and commonly in tournaments, that allows players to ensure load times are roughly equivalent.

Wingy Replaces Claw with Wings when you pick it up. Used for the Wingy format. If you have this installed and are playing non-wingy formats, remember to disable the mod before starting!

BingoGoalPackSpleengy An alternate goalpack for bingy, with automarking for a handful of additional goals.

Cosmetic Mods!

Finally, some cosmetic mods that are allowed, but not required, for bingy.

CustomKnight Allows you to have custom textures for the knight, as well as options for other sprites. And if you're looking for skins, Colette's skin list and HKSkins have a huge selection of over 200 skins that you can use with Custom Knight!

GrubTrain Grubs will follow you around once you have collected them! Be sure to turn off "grub tricks" in settings, as that is not bingy legal.

DeathCounter A death and hit counter to see how many times you failed. (We're still proud of you though.)

TransTrans Changes the colours of Tiktiks to be the trans flag.
