Bingy Skips to Know

BINGY

This page contains a list of bingy-relevant skips and movement you may want to learn. These are sorted into categories based on relevance and purpose, though there isn't really any particular order to the way skips are listed within those categories. See Gori's DebugMod Savestates for Bingy practice made for skips & movement practice for use with DebugMod. Please note that Slopeballs/Stallballs, are currently banned, so if you intend to use them in a match, clear this with your opponent(s) beforehand.
