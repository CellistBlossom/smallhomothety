Bingo boards for races!

Races using bingo boards are common in HKSC to add more depth to the race. Bingosync is the generator that we use to generate 5x5 bingo boards for formats such as blackout or 3-lines. Byngosink has similar options, but also includes generators for the more complex exploration or get to the other side formats, which Bingosync doesn't support.

Most vanilla races use the "normal" generator, while randomizer races usually use the "Item Randomizer - No tiebreakers" for solo races, and the normal "Item Randomizer" board for team races. The DAB generator is usually used for Double Anti Bingo races. Most HKSC races use the password "eggu" for our races! For a list of randomizer formats, see the formats page.

On Bingosync, you can also create a custom bingo board by selecting "Custom (Advanced)" as your game instead of Hollow Knight. This should be pasted into the "Board" field as a JSON list of goals. Some members of the community have put together custom goal lists for this purpose. For example, this is the blackout generator we sometimes use! Currently, Byngosink doesn't have this feature, so you will need to use Bingosync to generate these boards.

Jamie's Byngosink has mostly obsoleted the exploration bingo site, but you may want to use it if you want to play the Roguelike format (which neither Bingosync nor Byngosink include) or want extra customization in the layout of your boards. Unfortunately this site does not include any "syncing" of goals, so you won't be able to see what other people have marked.
